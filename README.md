# QQ自动化系统

QQ群采集+合约返佣获客全流程自动化工具。

---

## 系统架构

```
QQ号导入 → 温启动养号 → 群采集(禁言过滤) → 自动排期 → 自动执行(通过NapCat)
                                                                   ↑
                                                              AI话术改写
                                                              自适应风控
```

## 快速开始（Windows）

### 第一步：安装Node.js
1. 双击 `install-nodejs.bat` → 浏览器打开 nodejs.org
2. 下载 LTS 版本，双击安装，一路"下一步"
3. 装完打开 CMD 输入 `node --version` 确认显示版本号

### 第二步：启动系统
1. 双击 `start.bat`
2. 浏览器自动打开 http://localhost:3456
3. 管理界面出现，开始使用

### 可选：打包成独立exe
双击 `build-exe.bat` → 等待编译完成 → 得到一个`QQ自动化系统.exe`
之后双击exe就能用，不需要装Node.js了

## 使用指南

### 1. 导入QQ号
进入 👤 账号池 → 批量导入 → 每行格式: QQ号,密码,备注
系统自动进入温启动: 僵尸3天→观察4天→试探7天→正常期

### 2. 配置话术
进入 💬 话术库 → 添加话术 → 可点"AI改写"生成10个版本

### 3. 采集目标群
进入 🎯 群采集 → 在QQ里搜群 → 把结果粘贴进来
系统自动过滤: 禁言群❌ 人数不足❌ 合格群自动入库

### 4. 连接NapCat
进入 ⚙️ 设置 → 配置NapCat地址(默认 http://127.0.0.1:3000)

### 5. 启动自动执行
进入 📊 总控台 → 点"启动引擎"
系统自动: 按温启动阶段→按日间分布→按随机间隔→自动发消息

### 6. AI配置(可选)
进入 ⚙️ 设置 → AI配置 → 填入 DeepSeek/通义千问/GLM-4 API Key
开启: 话术AI改写 / 跟进判断 / 自适应风控分析

## 功能一览

| 模块 | 功能 |
|------|------|
| 账号管理 | 批量导入、温启动四阶段、每日限额追踪 |
| 群采集 | 搜索结果导入、禁言过滤、人数过滤、合格入库 |
| 调度引擎 | 随机间隔(正态分布)、日间分布(30%/20%/50%)、图谱冲突检测 |
| 执行引擎 | 自动拉取任务、调NapCat执行、结果回写 |
| 自适应引擎 | 自动评估封号率、动态调整限额和间隔 |
| AI引擎 | 话术改写(10版本)、跟进判断、温启动分析 |
| QQ注册 | 5sim接码集成、半自动注册、自动入库 |
| 联系人漏斗 | 7阶段追踪(刚加好友→已聊过→感兴趣→已发链接→已注册→已交易→丢了) |

## 风控策略

- **温启动**: 僵尸3天→观察4天→试探7天→正常期, 逐级放量
- **随机间隔**: 正态分布, 60%短/30%中/10%长, 模拟真人节奏
- **日间分布**: 上午30%/下午20%/晚上50%, 避开凌晨
- **话术轮换**: AI改写不同版本, 规避文本指纹
- **冲突检测**: 同一目标不分配给多个号, 防图谱关联
- **自适应**: 封号率>20%自动收紧, >50%自动停止, <5%逐步恢复

## 自适应规则

```
封号率 > 50% → 立即停止所有操作
封号率 > 20% → 自动收紧限额(私信≤5/天, 间隔≥10分钟)
封号率 < 5%  → 逐步恢复正常参数
发送成功率低 → 自动减少发送量
```

## 模拟器全自动注册QQ号

### 前置条件
1. 安装雷电模拟器 (https://www.ldmnq.com)
2. 模拟器里安装QQ App
3. 配置5sim账号 (https://5sim.net) 用于收短信验证码

### 录制注册流程（只需跑一次）
```
1. 打开雷电模拟器 → 打开QQ App
2. 在系统里点"开始录制注册流程"
3. 手动注册一个QQ号（每一步在系统里点"已操作"记录步骤）
4. 录制完成后导出流程
```

### 全自动注册（后续）
```
1. 选择一个已录制的流程
2. 系统自动从5sim获取手机号
3. ADB控制模拟器自动操作QQ完成注册
4. 新QQ号自动导入账号池
```
